# Proyecto Week 03: Dashboard de Plataforma de Streaming

## Descripción
Dashboard interactivo para una plataforma de streaming de video. Muestra lista de contenidos, estadísticas clave y datos en tiempo real sobre visualizaciones y actividad.

### Dominio
Plataforma de streaming de video | Entretenimiento y Medios

## Estructura
- Lista de videos disponibles
- Estadísticas de visualizaciones, usuarios activos y porcentaje de ocupación
- Indicador de visualizaciones en tiempo real

## Decisiones técnicas
- React + TypeScript
- useEffect para efectos y polling
- Custom hook para fetch
- Mock de datos y simulación de API

## Cómo ejecutar
1. Instala dependencias (`npm install`)
2. Ejecuta el proyecto (`npm start`)
